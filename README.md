# 🍔 Burguer Clan - Cardápio Digital Responsivo

Interface de cardápio interativo desenvolvido para hamburguerias e estabelecimentos de delivery, focado em experiência do usuário (UX) e design adaptativo.

---

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e organizada.
* **CSS3:** Estilização personalizada com foco em *Dark Mode*.
* **Bootstrap 5:** Framework utilizado para garantir total responsividade através de sistemas de grids e componentes modernos.
* **Google Fonts:** Tipografia selecionada para alta legibilidade e estética profissional.

---

## 📱 Destaques do Projeto

* **Design Mobile-First:** Interface otimizada para smartphones, ideal para acesso via QR Code em mesas ou links de bio em redes sociais.
* **Componentização:** Uso eficiente de classes do Bootstrap (Cards e Navbars) para um código limpo e de fácil manutenção.
* **Interface Premium:** Estética desenvolvida para o nicho de gastronomia artesanal, utilizando contrastes de cores quentes (Dourado/Amarelo) sobre fundo escuro.

---

## 🛠️ Conceitos de ADS Aplicados

* **Responsividade:** Adaptação de layout para diferentes tamanhos de tela (Desktop, Tablet e Mobile).
* **Usabilidade (UI/UX):** Foco na hierarquia visual, destacando preços e itens principais para facilitar a conversão de vendas.
* **Front-end Moderno:** Implementação via CDN para carregamento otimizado de bibliotecas externas.

---

## 📋 Como visualizar

Este é um projeto de front-end estático. Para visualizar:
1. Faça o download dos arquivos.
2. Abra o arquivo `index.html` em qualquer navegador.
3. (Opcional) Acesse via **GitHub Pages** caso o link esteja configurado no repositório.
